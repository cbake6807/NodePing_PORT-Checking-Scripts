Installing xinetd

Ubuntu / Debian Users:

apt-get install xinetd

Configuration file location: /etc/xinetd.conf

RedHat / CentOS Users:

yum install xinetd

Configuration file location: /etc/xinetd.conf


Check Script location directory: /etc/xinetd.d/

sh | curl -I https://raw.github.com/cbake6807/NodePing_PORT-Checking-Scripts/master/install.sh
